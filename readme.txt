Displays information about television shows using the trakt.tv API.

== Description ==

Displays the show's:

* Poster thumbnail
* Description
* Top episodes according to the trakt.tv ratings system

== Installation ==

1. Upload the `tv-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place a widget in your sidebar.

== Frequently Asked Questions ==

= Where do I get an API key from? =

You can get a free API key from http://trakt.tv/api-docs

= Can I have multiple widgets with different TV shows? =

Yes. You can have as many widgets as you want.
